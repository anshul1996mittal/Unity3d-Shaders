# \# 🌊☁️ Stylized Water \& Cloud Shaders (Unity)

# 

# A simple collection of \*\*procedural shaders built in Unity\*\*.

# 

# This project currently includes:

# 

# \* 🌊 Stylized Water Shader

# \* ☁️ Procedural Cloud Shader

# 

# Both shaders are built \*\*from scratch using code (HLSL)\*\* and do not rely on textures.

# 

# \---

# 

# \# 🎯 What this project does

# 

# ```text

# Water → Waves + Depth Color + Foam + Fresnel  

# Clouds → Noise + Movement + Sky Gradient

# ```

# 

# These shaders can be used in:

# 

# \* Games

# \* Flight simulators

# \* Open-world environments

# 

# \---

# 

# \# 🌊 Water Shader

# 

# Features:

# 

# \* Wave animation using sine functions

# \* Depth-based color (shallow to deep)

# \* Soft foam effect

# \* Fresnel edge highlight

# \* Fully customizable

# 

# \---

# 

# \# ☁️ Cloud Shader

# 

# Features:

# 

# \* Procedural cloud generation (noise-based)

# \* Animated movement

# \* Soft cloud edges

# \* Sky gradient (horizon to top)

# \* Lightweight and texture-free

# 

# \---

# 

# \# 🚀 How to Use

# 

# 1\. Create a material

# 2\. Assign shader:

# 

# ```text

# CustomShader/Water/StylizedWater\_Final

# ```

# 

# or

# 

# ```text

# CustomShader/Sky/CloudShader\_Final

# ```

# 

# 3\. Apply to a Plane / Quad in your scene

#
